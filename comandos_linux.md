# Comandos Essenciais do Linux

## ls
- Lista o conteúdo de um diretório.
  ```bash
  # O comando 'ls' exibe todos os arquivos e pastas dentro do diretório atual.
  ls /home

# O comando 'pwd' exibe o caminho absoluto do diretório em que você está.
pwd

# O comando 'cd' permite que você navegue entre diretórios. 
# Por exemplo, 'cd /var/log' move você para o diretório '/var/log'.
cd /var/log

# O comando 'mkdir' cria um novo diretório com o nome fornecido.
mkdir novo-diretorio

# O comando 'rm' remove arquivos. Se o diretório não estiver vazio, 
# você pode usar 'rm -r' para remover diretórios recursivamente.
rm arquivo.txt

# O comando 'cp' copia arquivos ou diretórios de um lugar para outro.
# Neste caso, 'cp arquivo1.txt /home/user/' copia 'arquivo1.txt' para o diretório /home/user.
cp arquivo1.txt /home/user/
